ANTHILL: Aplicación de intercambio de habilidades y trabajos circular. Usando puntos como moneda de cambio.

Creado por Silvia Guerola Ardid y Ana Urrialde Gómez como proyecto de aplicación ExpressJS con Handlebars para el bootcamp de Desarrollo Web en la escuela Ironhack:

                   https://anthill-project.herokuapp.com/inicio 



Aplicación Express con apoyo de almacenamiento en la plataforma MongoAtlas y renderizado de vistas con el motor de plantillas Handlebars.js
y desplegada en Heroku.
